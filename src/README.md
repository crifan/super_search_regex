## 简介

详尽整理正则表达式的基本语法；正则的名称的由来；整理出一张脑图看懂和记住所有正则的语法；以及用图形象解释了环视断言的种类和含义；常见的正则的应用领域和具体应用案例；整理了正则的心得，包括如何看懂别的的正则和如何自己写出相对复杂的正则。以及整理了好用的调试正则的工具和在线网站。
